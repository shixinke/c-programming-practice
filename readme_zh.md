#simple-webserver
一个使用c实现的简单的web服务器

##更新记录

v0.0.1:
实现一个最基本的web server，通过浏览器可以访问，在浏览器上显示It works


##使用方法

1、下载源码，编译生成可执行文件

    gcc webserver.c -o webserver
    
2、执行可执行文件，运行webserver
    
    ./webserver 127.0.0.1  9669
    
3、在浏览器上输入:http://127.0.0.1:9669
    
在浏览器上查看如果有It works表示成功了
